# MaaPTCGP

MaaPTCGP 是 Pokemon Trading Card Game Pocket 日常自动化辅助项目。

当前版本支持启动游戏、领取礼物盒、每日开包、每日商店票、得卡挑战、对战和领取日常任务奖励。

## 使用前提

- 推荐使用 MXU 运行任务。
- 需要通过 ADB 连接安卓模拟器或安卓设备。
- 当前识别主要适配中文游戏界面。
- 默认资源用于日常运行，开发资源用于项目调试。

## 语言适配说明

MaaPTCGP 的 MXU 界面支持中文、英文等多种语言。当前识别主要适配中文游戏界面。

如果使用英文、日文或其他语言的游戏界面，可能会识别失败或任务卡住。遇到报错时，请先将游戏切换到中文界面后再尝试。

如果任务卡住或识别失败，请在 GitHub Issues 反馈，并尽量附上任务名、设备/模拟器、游戏语言和 MXU 调试日志。
