# MaaPTCGP

MaaPTCGP 是 Pokemon Trading Card Game Pocket 日常自動化輔助專案。

目前版本支援啟動遊戲、領取禮物盒、每日開包、每日商店票、得卡挑戰、對戰和領取日常任務獎勵。

## 使用前提

- 推薦使用 MXU 執行任務。
- 需要透過 ADB 連接安卓模擬器或安卓設備。
- 目前識別主要適配中文遊戲介面。
- 預設資源用於日常執行，開發資源用於專案調試。

## 語言適配說明

MaaPTCGP 的 MXU 介面支援中文、英文等多種語言。目前識別主要適配中文遊戲介面。

如果使用英文、日文或其他語言的遊戲介面，可能會識別失敗或任務卡住。遇到報錯時，請先將遊戲切換到中文介面後再嘗試。

## 目前任務

- 🚀啟動遊戲
- 📬禮物盒郵件
- 🎴拆開卡包
- 🎟️每日商店票
- 🔮得卡挑戰
- ⚔️對戰
- 📋日常任務
- 👆上滑發送

## 回饋問題

如果任務卡住或識別失敗，請在 GitHub Issues 回饋，並盡量附上任務名、設備/模擬器、遊戲語言和 MXU 調試日誌。
